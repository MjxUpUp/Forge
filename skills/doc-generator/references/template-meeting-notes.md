# 会议纪要模板

## 变量列表

**必填**：
- `meeting_title`：会议主题
- `date`：日期
- `attendees`：参会人

**选填**：
- `agenda`：议程
- `decisions`：决议
- `action_items`：待办（含负责人+截止）

## 章节结构

```
# {meeting_title} 会议纪要

**日期**：{date}
**参会人**：{attendees}

## 议程与讨论
{按议程逐项记录讨论要点}

## 决议
{达成的决定，每条明确}

## 待办事项
| 事项 | 负责人 | 截止 |
|---|---|---|
{action_items}

## 遗留问题
{未决问题 + 后续处理方式}
```

## 风格示例

简洁要点、决议必须明确（不含糊）、待办有负责人和截止。
